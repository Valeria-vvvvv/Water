## ⚠️ Важные замечания

### Настройка Supabase

**Перед деплоем необходимо:**

1. Получить URL и Anon Key из [Supabase Dashboard](https://supabase.com/dashboard/org/vxvvxpliirsqtldivvnl)
2. Обновить файл `.env.production`:
   ```
   VITE_SUPABASE_URL=ваш_проект_url_из_supabase
   VITE_SUPABASE_ANON_KEY=ваш_anon_ключ_из_supabase
   ```
3. Создать таблицу `contacts` в Supabase (используйте `supabase_contacts_table.sql`)

### Проверка после деплоя

1. Откройте https://waterdonetsk.ru/
2. Проверьте консоль браузера (F12) - не должно быть ошибок
3. Заполните форму доставки воды
4. Проверьте:
   - Данные сохраняются в Supabase
   - Приходит уведомление в Telegram
   - Пользователь видит сообщение об успешной отправке

## 🐛 Если что-то пошло не так

### Форма не отправляется

1. Проверьте консоль браузера на наличие ошибок
2. Убедитесь, что переменные окружения заполнены
3. Проверьте подключение к интернету

### Ошибки в консоли

1. Убедитесь, что загружена новая версия файлов
2. Очистите кэш браузера (Ctrl+F5)
3. Проверьте `.htaccess` файл

### Данные не сохраняются в Supabase

1. Проверьте настройки RLS в Supabase
2. Убедитесь, что таблица `contacts` существует
3. Проверьте переменные окружения в `.env.production`
