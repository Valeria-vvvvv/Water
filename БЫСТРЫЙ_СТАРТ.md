# 🚀 Быстрый старт: Настройка проекта

## ✅ Что сделано

Проект настроен для работы с:

- **Supabase** - база данных для хранения заявок
- **Telegram Bot API** - уведомления о новых заявках

## 📝 Что нужно сделать (5 минут)

### 1️⃣ Настроить Supabase

1. Открыть [Supabase Dashboard](https://supabase.com/dashboard/org/vxvvxpliirsqtldivvnl)
2. Выбрать проект
3. В **Settings** → **API** скопировать:
   - **URL** (Project URL)
   - **anon** public ключ (Project API keys)

### 2️⃣ Создать таблицу contacts

В Supabase Dashboard:

1. Перейти в **Table Editor**
2. Нажать **Create a new table**
3. Использовать SQL из файла `supabase_contacts_table.sql`
4. Убедиться, что RLS (Row Level Security) включен

### 3️⃣ Обновить переменные окружения

Открыть файл `.env` и заменить значения:

```env
VITE_SUPABASE_URL=ваш_проект_url_из_supabase
VITE_SUPABASE_ANON_KEY=ваш_anon_ключ_из_supabase
```

### 4️⃣ Перезапустить проект

```bash
npm run dev
```

## ❓ Проблемы?

### "Supabase environment variables are not set"

→ Проверить `.env` файл и перезапустить `npm run dev`

### "insert violates row-level security policy"

→ Проверить настройки RLS в Supabase

### Данные не сохраняются

→ Проверить консоль браузера на наличие ошибок
