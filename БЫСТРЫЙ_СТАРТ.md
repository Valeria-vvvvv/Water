# 🚀 Быстрый старт: Тематические изображения

## ✅ Что сделано

Реализована система автоматической загрузки тематических изображений по методу преподавателя:

- **Pexels API** - поиск фото (без CORS проблем!)
- **Firebase Storage** - хранение изображений
- **Автоматическая загрузка** - при первом просмотре или массово

## 📝 Что нужно сделать (5 минут)

### 1️⃣ Получить Pexels API ключ

1. Открыть https://www.pexels.com/api/
2. Нажать "Get Started"
3. Зарегистрироваться (можно через Google)
4. Скопировать API Key

### 2️⃣ Добавить ключ в .env

Открыть файл `.env` и добавить строку:

```env
VITE_PEXELS_API_KEY=твой-ключ-сюда
```

Если файла `.env` нет:

```bash
cp .env.example .env
```

### 3️⃣ Настроить Firebase Storage

1. Открыть https://console.firebase.google.com
2. Выбрать свой проект
3. Меню слева → **Storage**
4. Нажать **"Get Started"**
5. Выбрать **"Start in production mode"**
6. Выбрать регион: **europe-west1**
7. Нажать **"Done"**

### 4️⃣ Настроить правила доступа

В Firebase Console → Storage → вкладка **"Rules"**

Заменить содержимое на:

```
rules_version = '2';
service firebase.storage {
  match /b/{bucket}/o {
    match /images/{categoryId}/{imageId} {
      allow read: if true;
    }
  }
}
```

Нажать **"Publish"**

### 5️⃣ Перезапустить проект

```bash
npm run dev
```

### 6️⃣ Загрузить изображения

1. Открыть сайт в браузере
2. Открыть консоль (F12 или Cmd+Option+I)
3. Выполнить:

```javascript
// Проверить статус
await window.imageLoader.checkImagesStatus();

// Загрузить все изображения (~10-15 минут)
await window.imageLoader.loadAllImages();
```

```javascript

await window.imageLoader.checkImagesStatus();
```

Должна появиться таблица с процентом загрузки для каждой категории.

## ❓ Проблемы?

### "VITE_PEXELS_API_KEY не установлен"

→ Проверить `.env` и перезапустить `npm run dev`

### "window.imageLoader is undefined"

→ Перезагрузить страницу

### Изображения не загружаются

→ Открыть консоль (F12) и посмотреть ошибки


## 🎯 Что дальше?

После загрузки изображений они будут:

- Автоматически показываться на карточках товаров

```bash
cd functions
npm install
firebase deploy --only functions
```
